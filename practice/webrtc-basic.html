🎥 WebRTC 기본 연습

📚 학습 목표: WebRTC의 기본 흐름을 이해하고 직접 구현해보기

💡 사용 방법:
  1. 이 파일을 브라우저 2개에서 열기 (탭 2개 or 창 2개)
  2. 한쪽에서 "Offer 생성" 버튼 클릭
  3. 생성된 Offer를 복사해서 다른 브라우저의 입력창에 붙여넣기
  4. 다른 브라우저에서 "Answer 생성" 버튼 클릭
  5. 생성된 Answer를 복사해서 첫 번째 브라우저에 붙여넣기
  6. 양쪽에서 서로의 영상이 보이면 성공! 🎉
